# GIS 算法基础 学习笔记

基于《地理信息系统算法基础》（张宏、温永宁、刘爱利 等编著，科学出版社 2006）的学习仓库。

## 全书目录

### 第1章 算法设计和分析
- 1.1 概述
- 1.2 算法设计原则
- 1.3 算法复杂性的度量（时间复杂性、空间复杂性）
- 1.4 最优算法
- 1.5 算法的评价（运行时间估计、最坏/平均情况分析、平摊分析）

### 第2章 GIS算法的计算几何基础
- 2.1 维数扩展的9交集模型（DE-9IM）
- 2.2 矢量的概念（加减法、叉积）
- 2.3 折线段的拐向判断
- 2.4 判断点是否在线段上
- 2.5 判断两线段是否相交
- 2.6~2.9 矩形/圆包含判断
- 2.10 判断点是否在多边形内（射线法、转角法）
- 2.11~2.15 线段/折线/多边形/矩形/圆在多边形内的判断
- 2.16~2.18 点/线段/圆在圆内的判断
- 2.19~2.21 线段交点、直线与圆交点计算
- 2.22~2.31 中心点、垂线、平行线、延长、画圆、线段打断、前方/距离交会、极坐标作点

### 第3章 空间数据的变换算法
- 3.1 平面坐标变换（平移、比例、对称、旋转、错切、复合变换）
- 3.2 球面坐标变换
- 3.3 仿射变换
- 3.4 地图投影变换（兰勃特、墨卡托、高斯-克吕格、通用横轴墨卡托投影）

### 第4章 空间数据转换算法
- 4.1 矢量数据向栅格数据转换（点/线/面的栅格化）
- 4.2 栅格数据向矢量数据转换（矢量化基本步骤、线状/多边形栅格数据的转换）

### 第5章 空间数据组织算法
- 5.1 矢量数据的压缩（间隔取点法、垂距法和偏角法、道格拉斯-普克法、光栏法）
- 5.2 栅格数据的压缩（链式编码、游程长度编码、块式编码、差分映射法、四叉树编码）
- 5.3 拓扑关系的生成（基本数据结构、弧段预处理、结点匹配、建立拓扑关系）

### 第6章 空间度量算法
- 6.1 直线和距离（直线方程、点到直线的距离）
- 6.2 角度量算
- 6.3 多边形面积的量算（三角形、四边形、任意二维/三维平面多边形）

### 第7章 空间数据索引算法
- 7.1 B树与B⁺树
- 7.2 R树结构（R树定义、主要操作算法、R*树算法）
- 7.3 四叉树结构（常规四叉树、线性四叉树、Z曲线和Hilbert曲线算法）

### 第8章 空间数据内插算法
- 8.1 概述（几何、统计、空间统计、函数、随机模拟、确定性模拟、综合方法）
- 8.2 分段圆弧法
- 8.3 分段三次多项式插值法（三点法、五点法）
- 8.4 趋势面插值算法
- 8.5 反距离权重插值算法（IDW）
- 8.6 双线性插值算法
- 8.7 薄板样条函数法
- 8.8 克里金法（普通克里金、通用克里金）

### 第9章 Delaunay三角网与Voronoi图算法
- 9.1 概述
- 9.2 Voronoi图
- 9.3 Delaunay三角形
- 9.4 Voronoi图生成算法（半平面的交、增量构造、分治、减量、平面扫描）

### 第10章 缓冲区分析算法
- 10.1 概述
- 10.2 缓冲区边界生成算法基础
- 10.3~10.5 点/线/面缓冲区边界生成算法
- 10.6 多目标缓冲区合并算法

### 第11章 网络分析算法
- 11.1 概述
- 11.2 网络数据模型
- 11.3 路径分析算法（单源/单目标/单结点对间/多结点对间最短路径、次短路径）
- 11.4 最佳路径算法（最大可靠路径、最大容量路径）
- 11.5 连通性分析算法（Prim算法、Kruskal算法）
- 11.6 资源分配算法

### 第12章 地形分析算法
- 12.1 数字地面模型的生成算法（离散点DEM规则网格生成、不规则三角网TIN生成、DEM数据结构转换）
- 12.2 基本地形因子分析算法（坡面因子提取、坡度、坡向、坡形）
- 12.3 地形特征提取算法（特征点提取、山脊与山谷线提取）
- 12.4 通视分析算法（两点间可视性判断、可视域计算）

### 第13章 空间数据挖掘算法
- 13.1 概述
- 13.2 分类算法（决策树分类）
- 13.3 泛化规则算法（概念层次、面向属性泛化）
- 13.4 相关分析（两要素/多要素相关分析、关联规则算法）
- 13.5 回归分析（一元/多元线性回归、非线性回归）
- 13.6 系统聚类分析
- 13.7 判别分析（距离判别、费歇判别、贝叶斯判别）
- 13.8 主成分分析（PCA）

### 第14章 数据输出算法
- 14.1 概述（地图符号构成、几何特征、SVG描述模型）
- 14.2 点状地图符号的绘制（圆、椭圆、多边形、五角星）
- 14.3 线状地图符号的绘制（平行线、虚线、短齿线、铁路线、境界线）
- 14.4 面状地图符号的绘制
